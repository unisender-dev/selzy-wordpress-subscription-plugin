== Description ==
The plugin allows you to create a subscription form on your website and send user data to Unisender.

All data of your subscribers will be sent to your account in Unisender.

== Installation ==
1. Install the plugin in wordpress.
2. Generate Unisender API key: https://www.unisender.com/ru/support/api/common/api-key/.
3. Create new subscription form.
4. Configure the subscription field.
5. Connect it to unisender.
